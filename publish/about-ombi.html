Plex Requests uses a product called Ombi.
Ombi automatically gives users the ability to request content by themselves!

If a Movie or TV Show is missing, search for it via Requests and click request
If the movie/show is on the web, it will be grabbed and added to Plex - usually in a few minutes
If a low quality release, it will automatically be upgraded when better quality becomes available
If not available yet, it will get downloaded as soon as it's on the web

Using the Ombi Mobile App to request Movies & TV

If you use the free Ombi App to make requests, you can scan for available releases and launch plex direct from the App, 
get notifications hen its been found and downloaded, plus access to all other requests
